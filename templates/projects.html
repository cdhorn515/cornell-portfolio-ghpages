Mystery Word Game
Node JS, express, sessions

A hangman style game. Sessions used to store information, authentication used to check for duplicate letter guesses, non-character guesses, and more than one letter guessed at a time.

Completed Summer 2017

Gabble Chat
Node JS, Sequelize, express, sessions

Twitter-like express application. Users information and "gabs" are stored on separate tables on a Sequelize database and cross-referenced by use of foreign keys. Likes are stored on a through table. User can create a login, save a gab of up to 140 characters, and like gabs, as well as see a list of who has liked a gab. User can also delete their own gab.

Completed Summer 2017

Code Snippet
Node JS, MongoDB, crypto, sessions

Express application to store user-created objects of code on Mongo db. Encrypted hash stored in database instead of user password. User can save created code snippets to database and search by language or tags for snippets they created.

Completed Summer 2017
